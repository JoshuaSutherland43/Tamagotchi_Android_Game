## ğŸ”„ Fragment System

The game uses **Android Fragments** to manage different activities in the duck's life. Switching between fragments allows for modular management of each action that impacts the duckâ€™s well-being and stats:

- **Happy Fragment**: Default state showing a healthy duck.
- **Reading Fragment**: Improves the duckâ€™s knowledge and mood.
- **Sleep Fragment**: Lets the duck rest, reducing depression.
- **Eat Fragment**: Feeds the duck, improving mood and energy.
- **Bath Fragment**: Keeps the duck clean and happy.
- **End Fragment**: Triggered if the duck becomes too depressed.
